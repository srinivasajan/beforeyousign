## 1️⃣ Shareable Analysis Links

### Why Use It?
Share your contract analysis with:
- **Your lawyer** for professional review
- **Business partners** for collaborative decision-making
- **Friends/family** for advice without re-uploading

### How to Use:

1. **Complete your analysis** (upload and wait for results)

2. **Click the "Share" button** in the top-right corner (next to Export)

3. **Configure your link**:
   ```
   Password Protection: [optional - leave empty for no password]
   Expires In: [choose 1, 3, 7, 14, or 30 days]
   ```

4. **Click "Generate Share Link"**

5. **Copy and share the URL** with anyone

### Security Features:
- ✅ Unique 30-character URLs (impossible to guess)
- ✅ Optional password protection
- ✅ Auto-expiration (1-30 days)
- ✅ View-only access (recipients can't edit)
- ✅ Auto-cleanup of expired links

### Example Use Cases:

**Scenario 1: Consulting Your Lawyer**
```
1. Upload employment contract
2. Generate password-protected link (7 days)
3. Email link + password to lawyer
4. Lawyer reviews analysis without needing original file
5. Lawyer responds with specific feedback
```

**Scenario 2: Business Partner Review**
```
1. Upload vendor agreement
2. Generate link without password (3 days)
3. Share in Slack/Teams with partners
4. Team discusses specific clauses in meeting
5. Make informed decision together
```

## 2️⃣ Keyboard Shortcuts

### Why Use It?
Navigate 3x faster without touching your mouse! Perfect for:
- Power users who analyze contracts frequently
- Accessibility needs
- Multitasking workflows

### Full Shortcut List:

| **Key** | **Action** | **What It Does** |
|---------|------------|------------------|
| `Ctrl+K` | Contract Map | Opens visual clause navigator |
| `Ctrl+M` | Chat Toggle | Shows/hides AI assistant |
| `Esc` | Close | Closes any open modal/sidebar |
| `Ctrl+/` | Help | Shows this shortcut list |
| `1` | Quick View | Switches to summary mode |
| `2` | Deep Dive | Switches to detailed mode |
| `3` | All Risks | Shows all risk levels |
| `4` | Critical | Shows only critical risks |
| `5` | High+Critical | Shows high and critical |

### Visual Hints:
- **Hover** over floating buttons (bottom-right) to see keyboard hints
- **Look for badges** showing "K", "M", "?" on hover
- **Press Ctrl+/** anytime to see all shortcuts

### Pro Tips:
```
📌 Start with Ctrl+K to get an overview (Contract Map)
📌 Use Ctrl+M to ask questions as you review
📌 Press 4 to focus on critical risks first
📌 Use Esc to quickly close and return to analysis
📌 Press 1/2 to toggle between quick and detailed views
```

**Negotiation Help:**
```
❓ "What should I negotiate in section 3?"

```

- ✅ Powered by Google Gemini 2.0

- ✅ Message counter badge shows activity

## 4️⃣ Contract Map

### Why Use It?
Get a bird's-eye view of your contract structure and jump to any section instantly.

### How to Access:
- **Click** the sparkle icon (bottom-right)
- **Or press** `Ctrl+K`

### Features:
```
🗺️ Visual hierarchy of all clauses
🎨 Color-coded by risk level:
   • Red = Critical risk
   • Orange = High risk
   • Yellow = Medium risk
   • Green = Low risk
   
🔍 Click any clause to jump directly to it
📊 See section types (Terms, Rights, Obligations, etc.)
```

### Use Cases:

**Before Meeting:**
```
1. Open Contract Map (Ctrl+K)
2. Scan for red/orange items
3. Prepare questions about those clauses
4. Close map (Esc) and review details
```

**During Review:**
```
1. Reading clause 3, want to check related clause 7
2. Open map (Ctrl+K)
3. Click clause 7 to jump instantly
4. Compare terms side-by-side
```

## 5️⃣ Risk Filtering

### Why Use It?
Focus on what matters most - don't get overwhelmed by low-priority items.

### How to Use:

**With Mouse:**
- Click filter buttons above the analysis

**With Keyboard:**
- Press `3` for All Risks
- Press `4` for Critical Only
- Press `5` for High + Critical

### Workflow Example:
```
Step 1: Press 4 (Critical only)
   → Review deal-breakers first
   
Step 2: Press 5 (High + Critical)
   → Check concerning items
   
Step 3: Press 3 (All risks)
   → Full review for completeness
```

## 6️⃣ Progressive Disclosure

### Why Use It?
Choose your detail level based on time available and contract complexity.

### Modes:

**Quick View (Press 1):**
- Executive summary
- Key risk metrics
- Critical red flags only
- ~2-minute scan

**Deep Dive (Press 2):**
- Full clause-by-clause analysis
- Detailed plain language explanations
- All risks and recommendations
- ~10-minute thorough review

### When to Use Each:

| Situation | Mode | Why |
|-----------|------|-----|
| Initial triage | Quick View | Fast assessment |
| Contract under $1K | Quick View | Proportional review |
| High-stakes deal | Deep Dive | Thorough analysis |
| Pre-lawyer review | Deep Dive | Prepare questions |
| Time-sensitive | Quick View → Deep Dive | Triage then deep dive |

## 🎯 Recommended Workflow

### First-Time Analysis:
```
1. Upload contract
2. Wait for analysis (~60 seconds)
3. Check risk score (top of page)
4. Press 4 to see critical risks
5. Press Ctrl+K to view Contract Map
6. Press Ctrl+M to ask questions
7. Press 2 for Deep Dive mode
8. Click "Share" to send to advisor
```

### Quick Review (5 minutes):
```
1. Upload → Wait
2. Press 1 (Quick View)
3. Press 4 (Critical only)
4. Read summary
5. Press Ctrl+M and ask: "What are my biggest risks?"
6. Make go/no-go decision
```

### Thorough Analysis (20 minutes):
```
1. Upload → Wait
2. Press 2 (Deep Dive)
3. Press Ctrl+K to review structure
4. Press 3 (All Risks) and read carefully
5. Press Ctrl+M to ask specific questions
6. Export report for records
7. Create share link for lawyer review
```

## 🆘 Troubleshooting

### Share Link Not Working?
```
❌ Problem: "Link not found" error
✅ Solution: Links are temporary (in-memory storage)
           They reset when server restarts
           Production version will use database
```

### Keyboard Shortcuts Not Working?
```
❌ Problem: Pressing Ctrl+K does nothing
✅ Solution: 1. Make sure page has focus (click on it)
            2. Don't use shortcuts inside input fields
            3. Check if browser extension is conflicting
```

### Chat Not Responding?
```
❌ Problem: Chat shows loading forever
✅ Solution: 1. Check API key in .env.local
            2. Restart dev server (Ctrl+C, npm run dev)
            3. Check browser console for errors
```

### Map Won't Open?
```
❌ Problem: Contract Map button does nothing
✅ Solution: Make sure analysis completed successfully
            Check for errors in browser console
```
